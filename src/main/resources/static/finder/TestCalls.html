Examples

Main Hub Finder About Response Schema Error Codes

Τα παρακάτω responses προκύπτουν ύστερα από επιτυχείς/ολοκληρωμένες κλήσεις του roleFinder:

Example 1.

Ο χρήστης gunetdemo δεν βρέθηκε στις βάσεις δεδομένων.

                
{
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 000,
  "message": "gunetdemo not found in any Database"
}

Example 2.

Ο χρήστης gunetdemo είναι μόνο φοιτητής.

                
{
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 100,
  "message": "gunetdemo found",
  "Roles" : [
    "Student"
  ]
}

Example 3.

Ο χρήστης gunetdemo έχει και τις 3 πανεπιστημιακές ιδιότητες.

                
{
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 111,
  "message": "gunetdemo found",
  "Roles" : [
    "Student",
    "Member of the Teaching Staff",
    "Associate"
  ]
}
