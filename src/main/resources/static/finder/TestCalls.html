Main Hub Finder About Response Schema

Τα παρακάτω responses προκύπτουν ύστερα από επιτυχείς/ολοκληρωμένες κλήσεις του roleFinder:

Example 1.

Ο χρήστης gunetdemo δεν έχει ενεργό ρόλο.

                
{
 "Response code": 200,
 "Message": "A Person with that loginName was not found."
}

Example 2.

Ο χρήστης gunetdemo είναι μόνο φοιτητής.

                
{
    "Response code": 100,
    "Message": "Person with given loginName was found.",
    "Affiliations" : [
     "student"
    ]
}

Example 3.

Ο χρήστης gunetdemo έχει πολλαπλές πανεπιστημιακές ιδιότητες, με την κύρια ιδότητα του να είναι μέλος προσωπικού.

                
{
    "Response code": 100,
    "Message": "Person with given loginName was found.",
    "Affiliations" : [
     "student",
     "staff",
     "faculty"
    ],
    "primaryAffiliation" : "staff"
}
