Examples

Main Hub Validator About Response Schema Error Codes

Τα παρακάτω responses προκύπτουν ύστερα από επιτυχείς/ολοκληρωμένες κλήσεις του loginNameValidator:

Example 1.

Αίτηση δημιουργίας λογαριασμού η οποία δεν επιστρέφει κανένα conflict βάσει αναζήτησης με το loginName. Επιπλέον ισχύει το 1 απο τα 2 ενδεχόμενα για τον αιτούμενο με αυτό το Α.Μ.Κ.Α.:

  • Είναι ο πρώτος λογαριασμός που δημιουργεί.
                
{
  "ssn": "12312312313",
  "ssnCountry": "GR",
  "birthDate": "19770823",
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 120,
  "message" : "No Conflicts Found, 12312312313-GR combination not found in any Database"
}
  • Θέλει να ξαναχρησιμοποιήσει το συγκεκριμένο loginName που του έχει ήδη αποδωθεί.
                
{
  "ssn": "12312312312",
  "ssnCountry": "GR",
  "birthDate": "19770823",
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 100,
  "message" : "No Conflicts Found"
}

Example 2.

Αίτηση δημιουργίας λογαριασμού η οποία δεν επιστρέφει κανένα conflict βάσει αναζήτησης με το loginName. Όμως για τον αιτούμενο με αυτά τα προσωπικά στοιχεία έχει καταχωρηθεί ήδη διαφορετικό userName.

                
{
  "ssn": "12312312312",
  "ssnCountry": "GR",
  "birthDate": "19770823",
  "loginName": "test",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 110,
  "message" : "No Conflicts Found, 12312312312-GR is already paired with at least 1 loginName",
  "personPairedWith": [
    "gunetdemo"
   ]
}

Example 5.

Αίτηση δημιουργίας λογαριασμού με ολοκληρωμένα στοιχεία, του οποίου όμως το loginName εντοπίστηκε ότι έχει ήδη καταχωρηθεί στον Data Server χωρίς να έχει ακολουθηθεί η τυπική διαδικασία δημιουργίας λογαριασμού.

                
{
  "ssn": "12312312312",
  "ssnCountry": "GR",
  "birthDate": "19770825",
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 300, 
  "message" : "gunetdemo already exists while not following the typical DS Account generation procedure"
}

Example 6.

Αίτηση δημιουργίας λογαριασμού με ολοκληρωμένα στοιχεία, του οποίου όμως για το loginName ενω υπάρχει τουλάχιστον μία εγγραφή σε κάποιο source, δεν βρέθηκαν επαρκή στοιχεία ώστε να μπορούμε να πούμε με σιγουριά ότι πρόκειται για το ίδιο άτομο (1 ή περισσότερα απο τα πεδία SSN, SSNCountry είναι NULL παντού).

                
{
  "ssn": "12312312312",
  "ssnCountry": "GR",
  "birthDate": "19770825",
  "loginName": "gunetdemo",
  "institution": "uoa"
}

To response που θα επιστραφεί:

                
{
  "Response code" : 310, 
  "message" : "A primary identifier given in the Request (ssn or ssnCountry) was never matched by any Record that is paired with 'gunetdemo'. We can not safely assume they are the same person."
}
