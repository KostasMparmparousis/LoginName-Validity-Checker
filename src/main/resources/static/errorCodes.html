Error Codes & Examples

Main Hub Validator Suggester Finder

Error Codes

  • 400:Missing attribute in JSON Request.
  • 401:Missing connection details for the institution given.
  • 500:Wrong/Invalid connection details for the given institution's Views and Data Server.

Error Examples

Example 1.

(The attribute "institution" was not provided in the JSON request)

                
{
  "Response code" : 400,
  "message": "No institution provided"
}

Example 2.

(Connection Details for uoa were not found)

                
{
  "Response code" : 401,
  "message" : "Could not connect to 'uoa'"
}

Example 3.

(Connection Details for the SIS were incorrect)

                
{
  "Response code" : 500,
  "message" : "Could not connect to the SIS"
}
